### 运行方式 ###

	java -jar .\CoMarkdown.jar

### 输出 ###

	[INFO] 		info开头的为日志

### 功能 ###

+ File菜单项：主要是打开和保存markdown文件
	+ Open			打开markdown文件
	+ Save			保存markdown文件
	+ Close			关闭markdown文件
	+ Export html	导出html文件
	+ quit			退出

+ Network菜单项：
	+ Host			作为服务器[一般在2332端口]
	+ Stop Host		停止服务
	+ Connect		作为客户端连接服务器
	+ Disconnect	客户端停止连接
+ Help：
	+ help...		浏览器链接至markdown文件
